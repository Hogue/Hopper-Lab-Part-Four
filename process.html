Our Process

Feature Branch Methodology

  1. Starting on our project
    1. Pull origin master from remote repo to your local master branch
    2. Create a branch for your new feature
    3. Code code code
  2. When Code is Ready:
    1. Checkout master and pull master from remote repo
    2. Checkout feature branch and `rebase master`
    3. Push local feature branch to origin feature branch (remote repo)
    4. Submit Pull Request to merge with Master branch in origin repo
  3. Once Merge is Complete
    1. Checkout master
    2. git pull origin master
    3. Go to Step 3
  4. Drink!
